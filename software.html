Software

In mijn opleiding ga ik 4 coderings programmas leren.

HTML & CSS

HTML (de Hypertext Markup Language) en CSS (Cascading Style Sheets) zijn twee van de kerntechnologieën voor het bouwen van webpagina's. HTML biedt de structuur van de pagina, CSS de (visuele en auditieve) lay-out, voor verschillende apparaten.

Javascript

JavaScript (vaak afgekort tot JS) is een lichtgewicht, geïnterpreteerde, objectgeoriënteerde taal met eersteklas functies, en is vooral bekend als de scripttaal voor webpagina's, maar wordt ook in veel niet-browseromgevingen gebruikt. Het is een op prototypen gebaseerde scripttaal met meerdere paradigma's die dynamisch is en objectgeoriënteerde, imperatieve en functionele programmeerstijlen ondersteunt.

SQL

Wat is SQL? SQL is Structured Query Language, een computertaal voor het opslaan, manipuleren en ophalen van gegevens die zijn opgeslagen in een relationele database. SQL is de standaardtaal voor relationeel databasesysteem.

PHP

PHP is een server-side scripttaal die is ingebed in HTML. Het wordt gebruikt om dynamische inhoud, databases, het volgen van sessies te beheren en zelfs hele e-commercesites te bouwen. Het is geïntegreerd met een aantal populaire databases, waaronder MySQL, PostgreSQL, Oracle, Sybase, Informix en Microsoft SQL Server.
